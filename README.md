# Tarefas-de-Laboratório
Tarefas de Laboratório da disciplina de Fundamentos de Programação, Engenharia de Computação UFC.

## Ambiente de compilação (em ajuste para exercícios antes do 5):
Os exercícios do beecrowd foram submetidos e avaliados no ambiente padrão da plataforma, utilizando o compilador C99.

Além disso, os códigos também foram testados localmente com a seguinte linha de compilação:

```bash
gcc -std=c89 -pedantic-errors -Wall -Wextra arquivo.c -o programa
```
